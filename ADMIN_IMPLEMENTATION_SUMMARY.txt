🔌 RUTAS API CREADAS
═════════════════════════════════════════════════════════════════════════════════

GET    /api/admin/users                  → Obtener usuarios (paginado)
POST   /api/admin/users                  → Crear usuario
PUT    /api/admin/users/:id              → Actualizar usuario
DELETE /api/admin/users/:id              → Eliminar usuario
PUT    /api/admin/users/:id/role         → Cambiar rol
POST   /api/admin/users/:id/reset-password → Reset contraseña
GET    /api/admin/logs                   → Activity logs (últimos 30 días)
GET    /api/admin/stats                  → Estadísticas y KPIs

🚀 INSTALACIÓN (5 MINUTOS)
═════════════════════════════════════════════════════════════════════════════════

PASO 1: Registrar rutas en Express (app.js)
  const adminRoutes = require('./backend/routes/admin');
  app.use('/api/admin', requireAuth, adminRoutes(supabaseClient));

PASO 2: Importar admin.js en index.html
  <script src="frontend/admin.js"></script>

PASO 3: ¡Listo!
  - Login como admin
  - Aparecerá "🔧 Admin Panel" en el sidebar
  - Click para acceder

💾 REQUISITOS DE BASE DE DATOS
═════════════════════════════════════════════════════════════════════════════════

TABLA: usuarios
  id                UUID PRIMARY KEY
  email             VARCHAR UNIQUE NOT NULL
  nombre            VARCHAR NOT NULL
  role              VARCHAR (valores: admin, supervisor, vendedor, bodeguero, cliente)
  activo            BOOLEAN DEFAULT true
  created_at        TIMESTAMP DEFAULT now()

TABLA: audit_logs
  id                UUID PRIMARY KEY
  usuario_id        UUID REFERENCES usuarios(id)
  evento            VARCHAR NOT NULL
  tabla             VARCHAR NOT NULL
  fila_id           VARCHAR
  detalles          JSONB
  timestamp         TIMESTAMP DEFAULT now()
