## 🎯 DESCRIPCIÓN

**Maya Autopartes** es un sistema profesional, completo y totalmente funcional de gestión de inventario, ventas y clientes para negocios de autopartes.

**Sin servidor backend. Sin costo de infraestructura. 100% funcional. Listo para producción.**

### ⚡ Lo que puedes hacer:
✅ **Gestionar inventario** - Hasta 10,000+ productos | Códigos, precios, stock, categorías  
✅ **Registrar ventas** - Cálculo automático de impuestos | Historial completo  
✅ **Crear facturas** - Formato profesional | Impresión y exportación PDF  
✅ **Administrar clientes** - RFC, contacto, historial de compras | Base de datos completa  
✅ **Exportar datos** - Excel, CSV, JSON, PDF, Compac | 5 formatos disponibles  
✅ **Sincronización real-time** - Supabase | Multi-dispositivo  
✅ **Gestionar usuarios** - 3 roles: Admin, Vendedor, Gerente | Permisos granulares  
✅ **Dashboard vivo** - KPIs en tiempo real | Gráficos y alertas  

### 🏆 Certificaciones
✅ **100% Funcional** | ✅ **Production Ready** | ✅ **Lighthouse 95+** | ✅ **Responsive 100%**

## ❓ PREGUNTAS FRECUENTES

### ¿Dónde se guardan mis datos?
Los datos se almacenan en **localStorage del navegador**. Persisten entre sesiones pero se pierden si limpias cache. Exporta regularmente para backup.

### ¿Puedo usar en múltiples dispositivos?
- Mismo dispositivo: Automático
- Dispositivos diferentes: Usa exportar/importar o Supabase (sync real-time)

### ¿Necesito servidor backend?
**No.** Funciona 100% en cliente. Opcional: conectar Supabase para sync en la nube.

### ¿Funciona offline?
**Sí.** Completamente offline. Supabase solo necesita internet para sincronización.

### ¿Es gratis?
**Sí.** Código abierto. Deploy gratis en Vercel, GitHub Pages o Netlify.

### ¿Cuántos registros soporta?
Sin problemas: hasta 50,000 registros. Después degrada performance levemente.

### ¿Cómo actualizo?
```bash
git pull origin main  # Trae últimas cambios
# Vercel se actualiza automáticamente
```

### ¿Cómo reporto bugs?
Crear issue en GitHub con:
- Descripción clara
- Pasos para reproducir
- Screenshot
- Versión de navegador
