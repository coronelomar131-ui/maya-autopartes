# 🚗 Maya Autopartes - Sistema de Gestión

Sistema completo de gestión para seguros y autopartes con dashboard, ventas, almacén, clientes, usuarios y facturas.

## ✨ Características

### 📊 **Módulos Principales**
- **Dashboard** - KPIs y estadísticas en tiempo real
- **Ventas** - Registro y seguimiento de ventas
- **Almacén** - Gestión de inventario de productos
- **Clientes** - Base de datos de clientes
- **Usuarios** - Gestión de usuarios del sistema
- **Facturas** - Generación de facturas profesionales
- **Exportar** - Exportación a CSV, JSON y Compac

### 🔐 **Seguridad**
- Login con usuario y contraseña
- Roles de usuario (Admin, Vendedor, Gerente)
- Sesiones persistentes

### 📈 **Funcionalidades**
- CRUD completo (Crear, Leer, Actualizar, Eliminar)
- Búsqueda y filtrado avanzado
- Generación de facturas con impresión
- Exportación de datos
- Diseño responsive (móvil, tablet, desktop)

## 🚀 Inicio Rápido

### Opción 1: Local
1. Descargar `maya-autopartes.html`
2. Abrir en navegador (Chrome, Firefox, Safari, Edge)

### Opción 2: En la Nube (Netlify)
1. Ir a https://netlify.com
2. Arrastrar el archivo `maya-autopartes.html`
3. ¡Listo! Tu sitio estará en línea

## 💾 Almacenamiento de Datos

- Los datos se guardan en **localStorage** del navegador
- No requiere backend ni base de datos
- Funciona completamente offline
- Los datos persisten entre sesiones

- **Colores de marca**: Navy (#1E4F7A) + Teal (#3BBCD4)
