🛒 MercadoLibre Integration - Demo & Examples

1️⃣ Cargar Scripts

Agregar los siguientes scripts en tu `index.html` antes de ``:

<!-- MercadoLibre Integration Scripts --> <script src="api/mercadolibre-sync.js"></script> <script src="api/meli-mapper.js"></script> <script src="api/meli-ui-integration.js"></script>
✓ Orden importante: mercadolibre-sync.js debe ir primero (contiene configuración base)

2️⃣ Inicializar en el HTML

Dentro de `DOMContentLoaded` o al cargar la app:

document.addEventListener('DOMContentLoaded', () => { // Inicializar MercadoLibre if (typeof window.MELI !== 'undefined') { window.MELI.init(); console.log('✅ MercadoLibre initialized'); } });

3️⃣ Agregar Sección en Modal de Configuración

En tu función `setupConfigModal()` en `ui.js`:

function setupConfigModal() { // ... código existente ... // Agregar sección MercadoLibre if (typeof window.MELI_UI !== 'undefined') { const meliSection = window.MELI_UI.createConfigSection(); const configContent = document.getElementById('configContent'); configContent.innerHTML += meliSection; // Actualizar UI window.MELI_UI.updateStatusUI(); } }
Output: Se mostrará: ┌────────────────────────────────┐ │ MercadoLibre Integration │ ├────────────────────────────────┤ │ Client ID: [____________] │ │ Client Secret: [______] │ │ [🔗 Conectar con ML] │ │ │ │ Estado: ❌ Desconectado │ └────────────────────────────────┘

4️⃣ Agregar Dashboard

En tu página de Home o Almacén:

function setupDashboard() { // ... código existente ... // Agregar dashboard MELI if (typeof window.MELI_UI !== 'undefined') { const dashboard = window.MELI_UI.createDashboard(); const mainContent = document.querySelector('.main-content'); mainContent.insertAdjacentHTML('beforeend', dashboard); // Actualizar cada 5 segundos setInterval(() => { window.MELI_UI.updateDashboard(); }, 5000); } }
Output: Se mostrará: ┌─────────────────────────────────┐ │ 🔄 MercadoLibre Sync │ ├─────────────────────────────────┤ │ Última sincronización: Nunca │ │ Próxima sincronización: - │ │ Estado: Inactivo │ │ │ │ [🔄 Sincronizar] [Configurar] │ └─────────────────────────────────┘

5️⃣ Integrar en Tabla de Inventario

Mostrar status de sincronización en cada fila:

function renderAlmacenRow(product) { const meliIcon = typeof window.MELI_UI !== 'undefined' ? window.MELI_UI.getStatusIcon(product) : '⚪'; const meliTooltip = typeof window.MELI_UI !== 'undefined' ? window.MELI_UI.getStatusTooltip(product) : ''; return ` <tr> <td>${product.nombre}</td> <td>${product.sku}</td> <td>$${product.precio}</td> <td>${product.stock}</td> <td title="${meliTooltip}">${meliIcon}</td> </tr> `; }

7️⃣ Ejemplos de Mapeo

Los módulos de mapeo convierten productos automáticamente
// Mapeo Local → MercadoLibre const localProduct = { id: 'prod_1', nombre: 'Puerta Delantera Izquierda', sku: 'MA-001', precio: 1250, stock: 50, categoria: 'puertas_autos' }; const meliProduct = window.MELI_MAPPER.mapLocalToMeli(localProduct); // Output: // { // title: "Puerta Delantera Izquierda", // category_id: "MLM6055", // price: 1250, // currency_id: "MXN", // available_quantity: 50, // seller_sku: "MA-001", // ... // }

9️⃣ Problemas Comunes

❌ "window.MELI is undefined"

Causa: Script no está cargado
Solución: Verificar que <script src="api/mercadolibre-sync.js"></script> está en index.html

❌ "No MELI authentication"

Causa: Usuario no ha conectado con MELI
Solución: Ir a Configuración → MercadoLibre → Conectar con MELI

❌ "Token expirado"

Causa: Token tiene 6 horas de vida
Solución: Automático (se refresca), o desconectar/reconectar

✅ "Stock no sincroniza"

Solución: Click "🔄 Sincronizar Ahora" o esperar 30 minutos
